# 🛍️ Einkaufen Shop Projekt

## 📌 Beschreibung
Ein einfacher Online-Shop, erstellt mit HTML und CSS. Optional erweiterbar mit Python Flask Backend.

## 📁 Struktur
shop-project/
├── index.html
├── style.css
└── app.py (optional)

## ✨ Features
- 🖼️ Produktanzeigen mit Bildern
- 📂 Kategorien-Seitenleiste
- 🔎 Suchleiste (UI / optional Backend)
- 🎯 Accordion-Produkte
- 📱 Responsives Design
- 🎉 Angebots-Banner

## 🎨 Design
- Gelb-Schwarz Farbstil
- Einfaches, modernes Layout
- Benutzerfreundlich und übersichtlich


## 🚀 Erweiterungen
- 🛒 Warenkorb
- 🗄️ Datenbank (SQLite/MySQL)
- 🔐 Login-System
- 📦 Produkt-API
- ⚡ Dynamische Suche

## 📌 Ziel
Lernprojekt für HTML, CSS und grundlegende Backend-Integration mit Python.
